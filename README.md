CRUD con Fetch, PHP OOP y MySQL
Integrantes
Luis Calderón, Luis Ortega

Descripción
Sistema CRUD completo para gestión de productos desarrollado con PHP Orientado a Objetos, MySQL y Fetch API. Permite realizar operaciones de crear, leer, actualizar y eliminar productos mediante una interfaz web moderna.

Características Principales
Programación Orientada a Objetos en PHP
Operaciones CRUD completas
Validaciones en cliente y servidor
Interfaz responsive con Bootstrap
Comunicación asíncrona con Fetch API
Notificaciones con SweetAlert2
Búsqueda en tiempo real


Tecnologías Utilizadas
Backend: PHP 8.3 con OOP
Base de Datos: MySQL con PDO
Frontend: HTML5, Bootstrap, JavaScript
Comunicación: Fetch API
Notificaciones: SweetAlert2

Funcionalidades Implementadas
Crear nuevos productos
Editar productos existentes
Eliminar productos con confirmación
Buscar productos en tiempo real
Validación de campos obligatorios
Mensajes de éxito y error
Interfaz responsive
